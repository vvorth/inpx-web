1.7.4 / 2026-07-02

1.7.3 / 2026-06-30

- Kobo теперь получает книги в привычном для него виде: при скачивании, отправке на почту и в Telegram формат `kepub` отдается как `.kepub.epub`. Регистр в имени формата больше не важен.
- Поиск по названиям стал терпимее к книжной пунктуации. Если название начинается с кавычки или скобки, книгу можно найти по самому тексту названия, без специальных символов в запросе.
- Список одинаковых названий аккуратнее работает с несколькими библиотеками: книги из разных источников больше не склеиваются в одну карточку только из-за похожих служебных полей.
- В админке появились быстрые показатели живости сервера: сколько WebSocket-запросов проходит, какие action тормозят и есть ли задержки event loop.
- Читалка лучше держит место при перестройке страниц после изменения шрифта, окна или отступов.
- Docker-сборка стала заметно быстрее: образ больше не тратит время на пересборку Node.js для `pkg`.
- У релизов появился нормальный источник заметок: GitHub Release берет описание прямо из `CHANGELOG.md`.
